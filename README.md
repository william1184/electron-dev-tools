# Comandos Utilizados na linha de comando do projeto

## Configurações Gerais
Instalar pacotes do package.json:
```
npm i
```

## Gerar setup
Rodar criador do setup do desktop
```
node make
 ou
npm make
```

## Rodar projeto
> !!!!!!Não Excluir o 01-config.svg!!!!!!
Rodar direto
```
node start
 ou
npm start
```
